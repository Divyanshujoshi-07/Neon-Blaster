# Neon-Blaster  🚀


Neon-Blaster is a futuristic space shooter game developed using HTML, CSS, and JavaScript. The game delivers an exciting arcade-style experience where players control a powerful neon spaceship and battle against incoming enemy ships in deep space.

The game features smooth spaceship movement, laser shooting mechanics, animated explosions, glowing cyberpunk visuals, and a dynamic starfield background that creates an immersive gameplay experience. Players must destroy enemy ships, avoid collisions, and achieve the highest score possible.

## ✨ Features

- Futuristic neon spaceship design
- Smooth player movement and controls
- Laser shooting system
- Enemy spaceship attacks
- Explosion particle effects
- High score saving using Local Storage
- Animated space background
- Realistic glow and lighting effects
- Game over and restart system
- Responsive canvas gameplay
- Clean and organized code structure

## 🎮 Controls

- **Arrow Left** → Move Left
- **Arrow Right** → Move Right
- **Spacebar** → Shoot Bullets
- **R Key** → Restart Game

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- HTML5 Canvas API
